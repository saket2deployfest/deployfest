# Deployfest Crowd Management Application

This is a monorepo containing the frontend and backend components of the Crowd Management Application.

## Folder Structure

- `/frontend`: Next.js frontend with Tailwind CSS and Firebase integration.
- `/backend`: FastAPI backend with YOLOv8 object detection integration.

## Getting Started

Refer to the individual README files in `/frontend` and `/backend` for detailed instructions on setting up and running each component.
